# dotfiles

### 1. 

```
sudo softwareupdate -i -a
xcode-select --install
```

### 2. Clone repo

```
mkdir Development && cd Development && mkdir mcclowes && cd mcclowes && git clone https://github.com/mcclowes/dotfiles.git
```

### 3. Run bootstrap

`sh bootstrap.sh`
